# Examples

Working examples to help you get started.

## Quick Start

| Example | Description | Time |
|---------|-------------|------|
| [01_train_tiny_model.py](01_train_tiny_model.py) | Train a tiny model from scratch | 5 min |
| [02_generate_code.py](02_generate_code.py) | Generate code with trained model | 1 min |
| [03_use_tools.py](03_use_tools.py) | Use tool calling capabilities | 1 min |

## Running Examples

```bash
# Train a tiny model (demo)
python examples/01_train_tiny_model.py

# Generate code (requires trained model)
python examples/02_generate_code.py

# Tool calling demo
python examples/03_use_tools.py
```

## Prerequisites

Each example checks for required files and provides instructions if something is missing.

For full training, see [docs/04-TRAINING-PIPELINE.md](../docs/04-TRAINING-PIPELINE.md).
